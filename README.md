# yup-find

```
NAME:
   find - search for files in a directory hierarchy

USAGE:
   find [PATH...] [EXPRESSION]

      Search for files in a directory hierarchy.

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --name value      base of file name matches shell pattern PATTERN
   --type value      file is of type TYPE (f=file, d=directory, l=link)
   --size value      file uses SIZE units of space
   --maxdepth value  descend at most LEVELS (a non-negative integer) levels (default: 0)
   --follow, -L      follow symbolic links (default: false)
   --help, -h        show help
```
